-What is a breakpoint?
A breakpoint stops the code from runnning and highlights where the breakpoint stopped the code. 

-How do you set a breakpoint?
Once in the developer tools window you can select sources and then highlight a line number to place an on-line breakpoint. You may also set a breakpoint by enabling the respectve breakpoint option in the breakpoints submenu
